# Simple TPS

Simple TPS is a Unity prototype of a simple third-person shooter game.

The project focuses on core TPS gameplay systems such as player control, combat basics, scene setup, prefabs, and gameplay scripting. It is intended as a small gameplay prototype rather than a full commercial product.

## What this project shows

- third-person character control
- gameplay scripting in C#
- scene and prefab-based Unity workflow
- basic combat / prototype game loop
- project structure for a small Unity game

## Tech stack

- Unity
- C#
- built-in Unity systems / packages used in the project

## Project structure

Main folders in `Assets`:

- `Scenes` – game scenes
- `Scripts` – gameplay logic
- `Prefabs` – reusable gameplay objects
- `Materials` – visual materials
- `PhysicsMaterials` – physics surface settings
- `Plugins` – third-party or helper assets

## Status

Prototype / learning project.

## How to run

1. Clone the repository
2. Open the project in Unity
3. Load the main scene from the `Assets/Scenes` folder
4. Press Play in the Unity Editor
